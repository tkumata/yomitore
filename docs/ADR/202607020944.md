# ADR: 成績レポート対象期間を90日に変更する

## Context

現在の成績レポートは、評価スコア集計と月次ヒートマップの対象期間を今日を含む直近30日としている。ユーザーは成績の保存・表示対象を90日に変更したい。

## Decision

- レポート画面で使用する対象期間を今日を含む直近90日に変更する
- 評価スコア集計と日次ヒートマップは同じ90日を使う
- 統計データの保存形式、読み込み形式、既存の集計 API は変更しない
- 画面表示と README の期間表記を90日相当に更新する

## Consequences

- レポート画面では、直近90日の評価スコアと日次ヒートマップを確認できる
- 既存の `stats.json` はそのまま利用できる
- 90日分の週列が表示対象になるため、狭い端末では既存の切り詰め処理の影響を受ける
