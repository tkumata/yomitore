# ADR: 成績レポート対象期間を180日に変更する

## 背景

現在の成績レポートは、評価スコア集計と日次ヒートマップの対象期間を今日を含む直近90日としている。ユーザーは成績の保存・表示対象を180日に変更したい。

## 決定

- レポート画面で使用する対象期間を今日を含む直近180日に変更する
- 評価スコア集計と日次ヒートマップは同じ180日を使う
- 統計データ構造と保存形式は変更しない
- 表示文言と README の期間表記を180日相当に更新する

## 結果

- レポート画面では、直近180日の評価スコアと日次ヒートマップを確認できる
- 既存の保存済み統計データはそのまま読み込める
- 180日分の週列が表示対象になるため、狭い端末では既存の切り詰め処理の影響を受ける
