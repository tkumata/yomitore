# yomitore ヘルプ

## 概要

yomitore は読解力を向上させるためのトレーニングツールです。
AI が生成した文章を読んで要約を作成し、その要約を AI が評価します。

## 使い方

### メニュー画面

- `↑/↓` または `j/k`: メニュー項目の選択
- `Enter`: トレーニング開始
- `r`: レポート表示
- `q`: アプリ終了

### トレーニング画面

#### 通常モード

- `i` または `Enter`: 入力モードに切り替え
- `↑/↓` または `j/k`: 原文をスクロール
- `Shift+↑/↓` または `Shift+j/k`: 評価結果をスクロール（評価結果表示時）
- `n`: 次のトレーニングへ（評価結果表示時）
- `r`: レポート表示/非表示
- `h`: このヘルプを表示/非表示
- `q`: アプリ終了

#### 入力モード

- `Esc`: 通常モードに戻る
- `Ctrl+S`: 要約を送信して評価を受ける
- 文字入力: 要約を入力
- `Backspace`/`Delete`: 文字削除
- `←/→`: カーソル移動
- `Home`/`End`: 行頭/行末へ移動
- `Enter`: 改行

### レポート画面

- `r`: レポートを閉じる
- `q`: アプリ終了

## 設定

### API キー

初回起動時に Groq API キーの入力が求められます。
環境変数 `GROQ_API_KEY` に設定することもできます。

## トレーニングのコツ

1. 原文をよく読んで、主要なポイントを把握する
2. 簡潔に要約する（文字数の目安: 原文の 1/3〜1/5 程度）
3. 重要な情報を漏らさない
4. 自分の言葉で書く

## 統計情報

- 月次レポート: 過去 30 日間のトレーニング結果
- 週次レポート: 過去 7 日間のトレーニング結果
- 成功率、トレーニング回数などを確認できます
