# Current Task: レポートヒートマップ週表示

## 状態

- Phase: 1
- Status: done
- Created: 2026-07-02 09:28

## 目的

月次レポートのヒートマップを、縦軸を曜日、横軸を週、セルを日にした表示へ変更する。

## 要件

- 横軸は週にする
- 横軸は左から古い週、右へ新しい週の順にする
- 週は日曜始まりとして扱う
- 縦軸は曜日にする
- 曜日は下から日、月、火、水、木、金、土の順にする
- 描画時の行順は上から土、金、木、水、火、月、日とする
- セルは対象期間内の1日を表す
- 対象期間外、またはその週に存在しない曜日のセルは `.` で表示する
- 対象期間内の日は Unicode block 文字で表示する
- `##` と `--` は使わない
- 色分けは既存の正解率ベースの段階を維持する
- 統計保存形式と集計 API は変更しない

## 実装予定

- `src/reports.rs` の `create_heatmap_without_badges` を週横軸・曜日縦軸の描画へ変更する
- 直近30日の開始日と終了日から表示対象の週列を算出する
- 対象期間外セルと日付が存在するセルを分けて描画する
- 既存の正解率ベースのセル色判定を再利用する
- ヒートマップ描画のユニットテストを週列前提へ更新する

- セルが対象期間内の1日を表す
- 対象期間外セルが `.` で表示される
- `##` と `--` が表示されない
